HTML

HTML and XHTML
HTML (Hypertext Markup Language) is used for structuring content on the web. XHTML is a stricter version of HTML that follows XML rules.
Difference between HTML4 and HTML5
HTML4 is an older version with limited features, while HTML5 is the latest version with new elements, attributes, and APIs for modern web development.
What is a domain
A domain is a unique name that identifies a website on the internet
What is a subdomain, main domain
A domain is a unique name that identifies a website on the internet eg: example.com.A subdomain is a subset of a main domain
Difference between HTTP and HTTPS
HTTP is an unsecured protocol, while HTTPS is a secure version that encrypts data using SSL/TLS.
Difference between semantic and non-semantic HTML:
Semantic HTML uses tags that provide meaning to the content eg: header, footer.Non-semantic HTML uses generic tags eg:div, span.
Difference between relative and absolute paths:
Relative paths are relative to the current file e.g., ./image.jpg.Absolute paths are complete URLs or file paths (eg: https://example.com/image.jpg).
